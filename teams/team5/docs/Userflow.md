User flow: User ziet als eerst de vraag: 'Hoe bekend bent u met het gebruiken van een AI-chatbot?'. 
Met daaronder de opties die gekozen kunnen worden: 
1. Niet bekend
2. Enigszins bekend
3. Erg bekend
Afhankelijker van de keus van de user wordt bepaald in welke mate de chatbot de user zal begeleiden in de rest van het gesprek.

Vervolgens wordt door de chatbot de vraag gesteld: 'Waaronder val je?'. De keuze opties zijn:
1. Ik ben patiënt of naaste
2. Ik ben algemeen publiek
3. ik ben een zorgverlener
4. ik ben een student of docent
5. ik ben een beleidsmaker
6. ik ben een onderzoeker of wetenschapper
7. Ik ben een journalist
8. Anders
Afhankelijk van de keus van de user wordt bepaald welk taalgebruik de chatbot zal gebruiken en welke bronnen gebruikt worden.

Hierna wordt door de chatbot de vraag gesteld: 'Wat is je vraag?'. Indien bij vraag 1 het antwoord 'Niet bekend' is gegeven, wordt hier de vraag gesteld: 'Schrijf in één zin duidelijk je vraag op.'
Om hierop antwoord te geven kan de user een antwoord typen.
